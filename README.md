## 📝 Installation

1. **Cloner le repo**
   ```sh
   git clone https://github.com/Lucas-Huerta/creative-dev.git
   cd creative-dev
   ```

2. **Installer les dépendances**
   ```sh
   npm install
   ```

3. **Démarrer le projet**
   ```sh
   npm run dev
   ```
